# Roboto Flex — вариативный специмен

Живая тестовая страница вариативного шрифта: печатаешь слово прямо в большой образец, крутишь оси ползунками, внизу собирается готовая строка `font-variation-settings`, которую можно скопировать.

Всё статическое, без сборки. Шрифты подгружаются с Google Fonts (Roboto Flex + IBM Plex Mono).

## Файлы

- `index.html` — разметка
- `style.css` — стили и токены темы (авто светлая/тёмная + ручной переключатель)
- `app.js` — ползунки осей, копирование, «случайно», сброс

## Локально

Просто открой `index.html` в браузере. Для загрузки шрифтов нужен интернет.

## GitHub Pages

1. Положи все файлы в корень репозитория и запушь:
   ```
   git add index.html style.css app.js README.md
   git commit -m "Variable font tester"
   git push
   ```
2. В репозитории: **Settings → Pages → Build and deployment**.
3. **Source: Deploy from a branch**, ветка `main`, папка `/ (root)`, сохрани.
4. Через минуту страница будет по адресу `https://<username>.github.io/<repo>/`.

Если кладёшь в существующий сайт-репозиторий, положи файлы в подпапку (например `vf/`) — тогда адрес будет `.../vf/`.

## Оси Roboto Flex

Основные: `wght` толщина, `wdth` ширина, `opsz` оптический кегль, `slnt` наклон.
Параметрические: `GRAD` насыщенность (меняет вес почти без сдвига набора), `XOPQ`/`YOPQ` толщина верт./гор. штрихов, `XTRA` ширина внутренних просветов, и группа `YT*` — высоты строчных, прописных, выносных и цифр.

Чтобы задать состояние в своём CSS, скопируй строку снизу страницы и добавь шрифт через `@font-face` или тот же линк на Google Fonts.
